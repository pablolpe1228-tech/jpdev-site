🚀 JPDev — Informática, Segurança & Desenvolvimento

Plataforma web institucional desenvolvida para apresentar os serviços da JPDev, integrando soluções em infraestrutura de TI, segurança eletrônica e desenvolvimento de software.

O projeto foi criado com foco em performance, responsividade e experiência do usuário, permitindo que clientes solicitem serviços diretamente pelo site.

🌐 Demonstração

Acesse o site publicado:

👉 jpdesenvolvimento.netlify.app

🧠 Sobre o Projeto

O JPDev foi desenvolvido como um site profissional para divulgação de serviços tecnológicos e captação de clientes.

A plataforma apresenta:

serviços de informática e manutenção

soluções de segurança eletrônica

desenvolvimento de software e automação

formulário inteligente para solicitação de serviços

O projeto também serve como portfólio técnico, demonstrando habilidades em desenvolvimento front-end e estruturação de aplicações web estáticas.

⚙️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

Tecnologia	Função
HTML5	Estrutura da aplicação
CSS3	Estilização e layout responsivo
JavaScript	Interatividade e lógica do site
Git	Controle de versão
GitHub	Hospedagem do código
GitHub Pages	Publicação do site
Formspree	Integração de envio de formulários
